# Retired Hands Empowering System

Academic web application developed as part of the university curriculum to support and empower retired individuals through digital interaction.

## Features
- Responsive frontend using HTML, CSS, Bootstrap, and JavaScript
- Backend developed with PHP
- MySQL database integration
- Form handling and validation
- CRUD operations

## Tech Stack
- HTML, CSS, Bootstrap, JavaScript
- PHP
- MySQL

## Project Type
Academic Mini Project

## Note
This project was developed for educational purposes.
